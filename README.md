# Control de Stock - Tecnobox Chile

Dashboard para análisis de descuadres de inventario y supervisión de stock en las sucursales Carahue 1 y Carahue 2.

## Características

- **Análisis por sucursal**: Pestañas separadas para cada tienda
- **Productos críticos**: Identificación de problemas sistemáticos
- **Indicadores de supervisión**: Evaluación de desempeño
- **Sin emojis**: Interfaz profesional para gerencia

## Datos Clave

- **Carahue 1**: 54 productos con problemas - Pérdida: $1,504,080
- **Carahue 2**: 289 productos con problemas - Pérdida: $8,530,450  
- **Total**: $10,034,530 en pérdidas identificadas
- **10 productos críticos** con fallas en ambas sucursales

## Uso

El dashboard permite al jefe evaluar:
1. Desempeño de supervisión por sucursal
2. Productos que requieren atención inmediata
3. Indicadores de control y descuadres
4. Análisis financiero del impacto

## Tecnología

- **Streamlit**: Dashboard interactivo
- **Pandas**: Procesamiento de datos  
- **Plotly**: Visualizaciones
- **Excel**: Fuente de datos
